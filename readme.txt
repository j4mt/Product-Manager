This is a simple OOP Product maintenance application with simply logging.

Demonstrates OPP Concepts:

1. Composition
2. Abstraction through interface definitions not abstract classes
3. Simply collections

Suggested enhancements:

4. no inheritance here, could be expanded to have different product types.
5. no polymorphism , could be expanded to use different products inherited from the same base product type.
6. Persistence service layer could be added.
7. H2, Mysql, SQLLite, could be used as a db.
8. ORM hibernate could be used to map Object entitys to a relation DB.
9. Spring MVC could be used to create a web front end, or just plain old jsp and servlets
10. Swing or SWT could be added also as a front end.
